# 🚀 Запуск на Daytona за 3 минуты

Проще всего — прямо в Terminal Daytona.

## ✅ Что нужно

- Аккаунт Daytona (https://daytona.io)
- Токен Telegram-бота от [@BotFather](https://t.me/BotFather)
- ✅ Всё! Больше ничего не нужно.

## 🎯 Запуск

### 1. Откройте Daytona Terminal

В Daytona IDE нажмите **Terminal** → откроется консоль.

Ты уже в папке проекта (по умолчанию).

### 2. Одна команда — всё запустится

Скопируй **целиком** эту команду (замени `YOUR_BOT_TOKEN` на реальный токен):

```bash
export BOT_TOKEN="YOUR_BOT_TOKEN" && bash daytona-deploy.sh
```

**Пример:**
```bash
export BOT_TOKEN="123456:ABCdefGHIjklmnopqrstuvWXYZ" && bash daytona-deploy.sh
```

### 3. Дождись окончания

Скрипт будет:
1. ✅ Установит зависимости (Node.js + Python)
2. ✅ Соберёт Next.js
3. ✅ Запустит веб-приложение
4. ✅ Запустит Telegram-бота

Должно занять **2-3 минуты**.

## 🌐 Доступ к приложению

Когда скрипт завершится, увидишь в Terminal:

```
✅ ПРИЛОЖЕНИЯ ЗАПУЩЕНЫ

🌐 Веб-приложение:   https://xxxxxxxx.daytona.app
🤖 Telegram-бот:     работает (в фоне)
```

**Открой ссылку в браузере** — должна загрузиться главная страница сайта.

## 🤖 Проверить Telegram-бота

1. Откройте Telegram
2. Напишите `/start` боту
3. Должен ответить

## 📝 Логи (если что-то не работает)

В том же Terminal выполни:

```bash
tail -f /tmp/web.log    # логи веб-приложения
tail -f /tmp/bot.log    # логи бота
```

Нажми `Ctrl+C` чтобы выйти из логов.

## 🔧 Если нужно перезапустить

```bash
# Остановить
pkill -f 'node server'
pkill -f 'python bot.py'

# Запустить заново
bash daytona-deploy.sh
```

## ✨ Готово!

Приложение работает:
- 🌐 Веб-панель на https://...daytona.app
- 🤖 Бот в Telegram

Пользователи могут:
1. Написать `/start` боту
2. Зарегистрироваться через `/register`
3. Войти на веб-панель
4. Создать сервер и запустить своего бота (если нужна функция управления)

---

💡 **Совет:** Если нужна полная функциональность с управлением Docker-контейнерами ботов пользователей, используй VPS вместо Daytona (см. INSTALL.md).

На Daytona функция управления ботами недоступна, только демо/разработка.
